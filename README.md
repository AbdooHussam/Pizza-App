# Pizza-App
تطبيق لمطعم بيتزا في سلطنة عمان .يقوم بعرض جميع الاطعمة المتوفرة وجميع الاصناف المختلفة بعد اختيار الوجبة يمكنك ان تحدد الكمية + الاضافات ثم ستذهب لمرحلة الدفع الالكتروني وتتبع الطلب + يوجد صفحة لعرض بيانات المستخدم ويمكنه تعديل جميع بيناته + يوجد صفحة لعرض جميع الطلبات السابقة
